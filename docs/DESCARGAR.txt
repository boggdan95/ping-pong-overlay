╔════════════════════════════════════════════════════════════════╗
║          🚀 INICIO RÁPIDO                                     ║
╚════════════════════════════════════════════════════════════════╝

1. Descarga al menos:
   ✓ index.html
   ✓ control.html

2. Abre en navegador:
   ✓ index.html en una pestaña (el overlay)
   ✓ control.html en otra pestaña (el control)

3. Prueba:
   ✓ Cambia nombres en control.html
   ✓ Cambia puntos con botones +/-
   ✓ Presiona tecla "1" o "2" en index.html
   ✓ Presiona tecla "S" para cambiar servicio

4. En OBS:
   ✓ Fuente → Browser Source
   ✓ URL: file:///ruta/a/index.html
   ✓ Aplica Chroma Key
